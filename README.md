# ArknightsMod
一个杀戮尖塔游戏的mod，由于做的不满意加上干员实在太多放弃了，现开源

# 特色
无新角色，战斗后有几率可选一名干员。干员通过能力牌召唤，可以使用事先在卡组界面选择的技能进行战斗。技力由你打出卡牌的消耗决定。

添加了怪物系列。每个篇章有一组怪物系列和最终boss。他们会随你打出卡牌的能量多次行动。
